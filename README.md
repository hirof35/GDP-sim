# 📊 GDP上昇・下降シミュレーター (GDP GUI Simulator)

マクロ経済学における**支出アプローチ（三面等価の原則）**の数式をベースに、政策介入や外部ショックが国内総生産（GDP）に与える動態をリアルタイムに可視化する、型安全な経済シミュレーターです。
<img width="1472" height="1027" alt="スクリーンショット 2026-05-30 072912" src="https://github.com/user-attachments/assets/e31a10aa-bec4-4161-b09a-34646ddd0888" />

$$GDP = C + I + G + (X - M)$$

## 🛠 特徴と経済学的な力学
- **乗数効果の再現**: 財政出動（$G$ の増加）を行うと、連鎖的に民間消費（$C$）も自動誘発される自己増幅フィードバックを実装。
- **トレードオフの制御**: 中央銀行の利上げ（金融引き締め）による投資（$I$）と消費（$C$）の抑制ダイナミズムを再現。
- **クリーンな構造分離**: 経済ロジック（`simulator.ts`）と、描画・ユーザー介入を司るビュー層（`main.ts`）を完全にデカップリング。

## 🚀 テックスタック
- **Language:** TypeScript
- **Bundler / Dev Server:** Vite (v8.x)
- **Visualization:** Chart.js (auto)
